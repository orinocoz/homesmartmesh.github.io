Integrated Software into the IoT Framework

  • C++ RF Gateway: RF Mesh network to MQTT
  • Mosquitto: MQTT Broquer
  • Mongodb: Document Database for complex structures data
  • InfluxDB: Timeseries Database for simple values over time
  • Grafana: Generic exploration of time series data
  • React+D3js: Highly customised frontend webapp
  • OpenHAB2: forwards connected devices to MQTT (Smart sockets,...)
  • NodeRed: Easy High level IoT rules

Binary RF Packets Format

  • Pid: Function Id (e.g. light, temperature,...)
  • CRC: 16 bit Error Protection
  • Broadcast: simple packet flood to all of the mesh listeners
  • Peer to peer: reliable end to end communication
  • Message: Handling of the acknowledge by the stack
  • Streaming: possible to drop the acknowledge for higher throughput
  • Time to live: configurable from the sender

Sensor Tag

  • Sensors: Temperature, Humidity, Pressure, Light, contact
  • Beacon: simple broadcast e.g. every minute

Communication Modes

  • Broadcast: simple flood to all listeners
  • Peer to peer: sender expects to listen to an ack flooded back
